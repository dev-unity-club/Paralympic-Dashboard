# Paralympic Games Analytics Dashboard — Power BI Report

This report provides a comprehensive analysis of Paralympic Games data spanning from 1960 to 2018, built for the International Paralympic Committee (IPC). It consists of four interactive dashboards:



### Overview Dashboard — 

High-level KPIs including 11K+ athletes, 272 events, 127 countries, and medal counts (gold, silver, bronze), with a world map of medals by country, a treemap by continent, and a medal trend line over the years.



### Country Dashboard — 

Deep-dive into National Paralympic Committee (NPC) performance, featuring medal trends across Games editions, a gold/silver/bronze breakdown, top 10 NPCs by total medals, and a detailed country-level medal table led by the USA, GBR, and CAN.



### Sport \& Event Dashboard — 

Analysis of medal distribution across sports, with a decomposition tree for drill-down by medal type, sport, and athlete; a pie chart of sport-wise share; and a bar chart of the top 10 sports by medal count (Athletics leading at 1,484).



### Country Comparison Dashboard — 

Side-by-side comparison of selected countries (e.g., Argentina vs. Brazil) with stacked bar charts, year-wise medal counts, and trend lines over time.



All dashboards include slicers for Games Season, Year range, Country, Sport, and Event for fully interactive filtering.
